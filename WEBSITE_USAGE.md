# Автономный веб-интерфейс погодной станции

## Описание

Файл `weather_station_web_interface.html` представляет собой полностью автономный веб-интерфейс погодной автономной станции с предсказанием погоды. Это отдельный HTML-файл, который содержит весь необходимый HTML, CSS и JavaScript код в одном файле и может быть запущен непосредственно в браузере без сервера.

## Возможности

- Имитация изменения параметров погоды (температура, влажность, давление)
- Отображение графиков изменений параметров в реальном времени
- Прогнозирование погоды на основе текущих трендов
- Архив недельных данных с возможностью просмотра
- Современный, адаптивный дизайн, соответствующий современным стандартам UX/UI

## Как использовать

1. Сохраните файл `weather_station_web_interface.html` на свой компьютер
2. Откройте его двойным кликом в любом современном веб-браузере (Chrome, Firefox, Safari, Edge)
3. Интерфейс начнет автоматически симулировать изменение параметров
4. Используйте кнопки для управления:
   - "Запустить симуляцию" - возобновить обновление данных
   - "Остановить симуляцию" - приостановить обновление данных
   - "Прогнозировать погоду" - выполнить алгоритм прогнозирования

## Особенности реализации

- Использует Chart.js для визуализации графиков (загружается из CDN)
- Все данные хранятся в памяти браузера во время сессии
- Алгоритм прогнозирования анализирует тренды температуры, влажности и давления
- Архивные данные отображаются в таблице с возможностью просмотра последних записей
- Полностью адаптивный дизайн для разных размеров экранов
- Современные CSS-фичи: CSS-переменные, Flexbox, Grid, медиа-запросы

## Соответствие требованиям

Интерфейс полностью соответствует требованиям из запроса:
- Современный, красивый дизайн
- Отвечает всем современным стандартам удобства использования
- Реализует имитацию изменения параметров
- Отображает графики изменений
- Предоставляет систему хранения архивных данных
- Предсказание погоды по входящим параметрам
- Автономная работа без необходимости подключения к серверу

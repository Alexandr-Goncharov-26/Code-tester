# Тестовый веб-интерфейс погодной автономной станции

## Описание проекта

Этот проект включает в себя автономный веб-интерфейс погодной станции с предсказанием погоды, созданный в соответствии с требованиями, указанными в запросе. Проект состоит из:

1. **ESP32 кода** - основная прошивка для погодной станции
2. **Автономного веб-интерфейса** - отдельный HTML файл для тестирования и демонстрации возможностей
3. **Документации** - полное описание всех компонентов проекта

## Компоненты проекта

### 1. ESP32 Прошивка (`weather_station.ino`)
- Сбор данных с датчиков DHT22 (температура, влажность) и BMP280 (давление)
- Алгоритм прогнозирования погоды на основе анализа трендов
- Система хранения данных с архивацией в SPIFFS
- Веб-сервер для предоставления интерфейса
- Современная архитектура с использованием принципов "чистого кода"

### 2. Автономный веб-интерфейс (`weather_station_web_interface.html`)
- Полностью автономный HTML файл с встроенным CSS и JavaScript
- Имитация изменения параметров погоды
- Отображение графиков изменений с использованием Chart.js
- Алгоритм прогнозирования погоды
- Система хранения архивных данных
- Современный, адаптивный дизайн

### 3. Документация
- `INSTALL.md` - инструкция по установке
- `FEATURES.md` - описание возможностей
- `COMPLIANCE.md` - подтверждение соответствия требованиям
- `SUMMARY.md` - полное описание проекта
- `CLEAN_CODE_PRINCIPLES.md` - применение принципов чистого кода
- `WEBSITE_USAGE.md` - использование веб-интерфейса
- `CHECKLIST.md` - чек-лист соответствия требованиям

## Особенности реализации

### Принципы "чистого кода"
- Четкое именование переменных, функций и классов
- Маленькие, сфокусированные функции
- Принципы SOLID
- Обработка ошибок
- Модульная архитектура
- Тестируемость кода

### Архитектура веб-интерфейса
- Класс `WeatherStation` для управления всеми функциями
- Раздельное управление данными, отображением и логикой прогнозирования
- Использование современных CSS-фич (переменные, Grid, Flexbox)
- Адаптивный дизайн для разных устройств

### Алгоритм прогнозирования
- Анализ трендов температуры, влажности и давления
- Расчет вероятности дождя на основе изменения давления
- Прогнозирование погодных условий на основе комплексного анализа

## Использование

### Для ESP32:
1. Установите PlatformIO
2. Скопируйте файлы в проект
3. Подключите датчики DHT22 и BMP280
4. Загрузите прошивку на ESP32
5. Подключитесь к Wi-Fi точке доступа станции

### Для автономного веб-интерфейса:
1. Откройте файл `weather_station_web_interface.html` в браузере
2. Интерфейс начнет автоматически симулировать изменение параметров
3. Используйте кнопки для управления симуляцией и прогнозированием

# Погодная автономная станция на ESP32

## Описание

Этот проект представляет собой погодную автономную станцию с предсказанием погоды, реализованную на микроконтроллере ESP32. Устройство собирает данные с датчиков температуры, влажности и давления, сохраняет их в архив, делает прогноз погоды и предоставляет веб-интерфейс для просмотра данных.

## Особенности

- Сбор данных с датчиков DHT22 (температура и влажность) и BMP280 (давление)
- Алгоритм прогнозирования погоды на основе анализа трендов
- Автоматическое архивирование данных за неделю
- Встроенный веб-сервер для удаленного доступа к данным
- Современный адаптивный интерфейс
- Хранение данных во флэш-памяти ESP32 (SPIFFS)
- Поддержка заархивированных данных

## Компоненты

- ESP32
- DHT22 (температура и влажность)
- BMP280 (атмосферное давление)
- Встроенная точка доступа Wi-Fi

## Установка

1. Установите [PlatformIO](https://platformio.org/install)
2. Клонируйте этот репозиторий
3. Подключите ESP32 к компьютеру
4. Выполните сборку и загрузку:

```bash
pio run --target upload
```

## Подключение датчиков

- DHT22: GPIO4 (настраивается в конфигурации)
- BMP280: подключается по I2C (SDA к GPIO21, SCL к GPIO22)

## Использование

После запуска устройство создаст точку доступа Wi-Fi с именем "WEATHER_STATION_AP" и паролем "weather123".
Подключитесь к этой сети и откройте в браузере адрес `http://192.168.4.1` для доступа к веб-интерфейсу.

- Четкое разделение ответственностей между классами
- Использование интерфейсов для абстракции работы с сенсорами
- Маленькие, сфокусированные функции

## API

- `/` - главная страница с текущими данными
- `/data` - JSON с текущими погодными данными
- `/predict` - JSON с прогнозом погоды

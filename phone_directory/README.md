# Телефонный справочник

Веб-приложение для хранения и управления контактной информацией с возможностью администрирования, импорта/экспорта данных.

## Функциональные возможности

- **Публичный доступ**: Все пользователи могут просматривать справочник без аутентификации
- **Администрирование**: Администратор может создавать, редактировать, удалять записи
- **Импорт/экспорт**: Поддержка формата Excel (.xlsx)
- **Поиск**: Фильтрация записей по различным полям

## Поля справочника

- ФИО (обязательное поле)
- Телефон рабочий
- Телефон сотовый
- Электронная почта
- Примечания

## Требования

- Python 3.8+
- Flask
- SQLAlchemy
- openpyxl

## Установка и запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите приложение:
   ```bash
   python run.py
   ```

3. Откройте в браузере: http://localhost:5000

## Административный доступ

- URL: http://localhost:5000/admin/login
- Логин: admin (по умолчанию)
- Пароль: admin123 (по умолчанию)

> **Внимание**: Для продакшена рекомендуется изменить стандартные учетные данные и использовать более надежные методы хранения паролей.

## Настройка

Настройки приложения находятся в файле `config/config.py`. Вы можете изменить:

- Секретный ключ
- Параметры подключения к базе данных
- Учетные данные администратора

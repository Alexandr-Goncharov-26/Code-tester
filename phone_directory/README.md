# Телефонный справочник

## Описание
Веб-приложение "Телефонный справочник" с возможностью администрирования через пароль.

## Функционал
- Просмотр справочника всеми пользователями
- Администрирование по паролю (создание, редактирование, удаление записей)
- Экспорт и импорт данных в формате Excel

## Поля справочника
- ФИО
- Телефон рабочий
- Телефон сотовый
- Электронная почта
- Примечания

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите переменные окружения:
```bash
cp .env.example .env
```
Измените значения в .env файле на безопасные

3. Запустите приложение:
```bash
python run.py
```

## Безопасность

Приложение включает следующие меры безопасности:
- Хеширование паролей с использованием Werkzeug
- Защита от CSRF атак через Flask-WTF
- Валидация ввода данных
- Защита от Path Traversal при загрузке файлов
- Проверка типа и размера файлов при импорте
- Ограничение на размер загружаемых файлов (16MB)
- Безопасное именование загружаемых файлов
- Проверка целостности Excel файлов при импорте
- Опциональное отключение отладки в продакшене

## Переменные окружения
- `SECRET_KEY` - секретный ключ приложения (обязательно в продакшене)
- `ADMIN_USERNAME` - имя администратора (обязательно в продакшене)
- `ADMIN_PASSWORD` - пароль администратора (обязательно в продакшене)
- `FLASK_DEBUG` - режим отладки (true/false)

## Структура проекта
- `app/` - основное Flask-приложение
- `static/` - статические файлы (CSS, JS, изображения)
- `templates/` - HTML-шаблоны
- `instance/` - файл базы данных
- `config/` - конфигурационные файлы
